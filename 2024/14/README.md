- operator override: so often I missed it in Java

- I didn't find a way to generalize class Vector, so that the following could be declared in one place, here with a simpliefied body:
```kotlin
data class VectorInt(val x: Int, val y: Int) {

    operator fun plus(other: VectorInt): VectorInt {
        return VectorInt(x + other.x, y + other.y)
    }
}

data class VectorDouble(val x: Double, val y: Double) {

    operator fun plus(other: VectorDouble): VectorDouble {
        return VectorDouble(x + other.x, y + other.y)
    }
}
```
e.g. as `data class VectorGen <T : Number> (val x: T, val y: T)` because of bouncing back from the inability to call `plus` operator on the Number (interface) level
